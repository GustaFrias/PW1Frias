Trabalho sobre JavaScript

JavaScript e Java são a mesma coisa?
R:Não, JavaScript e Java não são a mesma coisa. Embora os nomes sejam parecidos, eles são linguagens de programação diferentes com objetivos distintos. JavaScript é usado principalmente para desenvolvimento web, enquanto Java é uma linguagem de propósito geral.
Quando surgiu o JavaScript?
R:O JavaScript foi criado em 1995.
Quem foi o responsável pela criação do JavaScript?
R:Brendan Eich foi o responsável pela criação do JavaScript.
Qual o caminho trilhado pelo responsável pela criação do JavaScript, trata-se de alguma empresa ainda no mercado?
R:Brendan Eich criou o JavaScript enquanto trabalhava na Netscape, uma empresa pioneira na internet. Atualmente, Brendan Eich é o CEO da Brave Software, empresa conhecida pelo navegador Brave.
